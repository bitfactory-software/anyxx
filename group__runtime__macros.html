Detailed Description

Macros to reduce boilerblate for declaring/defining static runtime Data.

To enable some dynamic functionality (crosscast, open dispatch) there mus be some static data holding information to do this operations. Extra care must be taken in a dll scenario.

To reduce boilerplate code and Any++ suplies this macros
