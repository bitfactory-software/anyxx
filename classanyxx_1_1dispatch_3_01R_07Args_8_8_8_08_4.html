Detailed Description

template<typename R, typename... Args>
class anyxx::dispatch< R(Args...)>

Open dispatch method. Solves expression problem.

Template Parameters
R(Args..)Signature of the dispatch

The open dispatch is managed via a sigleton object of this class.

To declare and define a singleton use the ANY_SINGLETON and ANY_SINGLETON_DECLARE macros.

Each trait of the any tagged as virtual, must have the open dispatch enabled. To enabele the open dispatch you have to declare a struct named 'trait_name'_has_open_dispatch.

struct node_has_open_dispatch {};
ANY(node<>, , )
// open dispatch evaluate, returns int, dispatched via an any_node;
dispatch<int(virtual_<any_node<>>)> evaluate;
