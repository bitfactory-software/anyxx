Detailed Description

template<typename Value>
struct anyxx::by_val< Value >

Proxy to capture the dispatch target concrete to enable static dispatch A simple wrapper class over an object. Use '&' and 'const &' to capture by reference.

Usage:

  • Use the model map as a static customization point.
  • Use by_val<std::variant<...>> to unify customization points and member function-like invocation.
  • Use with vany_variant.
