Detailed Description

template<is_proxy Proxy, typename Trait>
class anyxx::any< Proxy, Trait >

The core class template to control dispatch for external polymorphism.

To control the behaviour, any provides two template parameter, Proxy and Trait. Imagine this as a combination af a std::any and some std::function's. With the Proxy tempalate param you control, wether this any behaves like a copying funcction, a move_only function, a ref function or if the target object is caputred cooncrete inside of any. With the Trait tempalte paramteter you specify the member functions of an captured object, witch can be invoked on this any.

Template Parameters
ProxySpecifies the lifteime of the captured object. anyxx provides by_val, cref, mutref .... All Proxy classes must conform to the is_proxy concept.
TraitSpecifies the functionality of this any. A class of this type is normaly provided via a TRAIT or ANY macro.
